# Coin Change — Research & Foundations

Fewest coins to make an amount — an unbounded-knapsack dynamic program and one of the canonical DP problems.

- **R. Bellman, *Dynamic Programming*, Princeton University Press, 1957.** The origin of dynamic programming — optimal substructure and overlapping subproblems, the paradigm this problem is built on.

**Why it matters here:** The DP builds each amount’s optimum from smaller amounts (optimal substructure + overlapping subproblems) — textbook Bellman dynamic programming.
